1. [URL-Encoding](https://developer.mozilla.org/en-US/docs/Glossary/percent-encoding) needed for spaces or special characters!

2. The default value of our icon is `/badge.svg`, which can be omitted from the URL.

3. Only `hex` color setting is supported.

4. Through [UptimeRobot](https://stats.uptimerobot.com/qLRJEIOg7Q) monitoring system normal operation.
